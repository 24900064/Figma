# Ex09 Event Registration Web Application
## Date: 12.05.2025

## AIM:
To design, develop and deploy a web application for event registration.

## DESIGN STEPS:

### Step 1:
Create a new frame.

### Step 2:

### Step 3:

### Step 4:

### Step 5:
